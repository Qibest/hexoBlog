Thread死锁问题

2016/07/08 カテゴリー Java

##线程死锁
当线程发生死锁的情况时，用eclipse很难找出真正的原因所在。需要借住jdk自带的“java VisualVM”，这个小程序在jdk的bin目录下面。
找到对应的线程。如发现有长时间“Monitor”状态的就证明有“死锁的情况”。需要分析CPU 和 Thread Dump 找到死锁的原因。
